  • [V3] God Mode + Ghost Enemy
  • [V3f] God Mode + Freeze Enemy

  • [V4] Inf MP

  • [V7] Instawin

  • s = 2x speed hack
  • ss = 4x speed hack
  • sss = 8x speed hack

  • p = Auto-parry

  • c = Crit

  • w = Weak enemy (Elemental Weak)
