Jenis Mod:

  • [V1]Mode Tuhan (kebal)
  • [V2] Automatik Parry / Crit / Enemy lemah
  • [V3] Mode Tuhan + Enemy Hantu
  • [V3f] Mode Tuhan + Enemy Beku

    • JANGAN GUNAKAN (v4-v7) dalam acara peringkat, dijamin kurang dari 1 jam terjejas banned
  • [V4] MP tak terhad
  • [V5] Mode Tuhan + MP tak terhad
  • [V6] Mode Tuhan + 1juta serangan kuasa
  • [V7] Memenangi mudah

  • h = 0.5 kelajuan hack

  • s = 2x kelajuan hack

  • p = Parry secara automatik

  • c = Crit(kritikal)

  • w = Enemy lemah (Kelemahan Element)

  • 2xx = versi aplikasi
